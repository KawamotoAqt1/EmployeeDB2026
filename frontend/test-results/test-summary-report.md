## 型定義の確認

### ProjectAssignment インターフェース

**確認事項**: workload プロパティの有無

コードレビューで確認した結果：
- ProjectDetail.tsx で `workload` が使用されている（line 53, 245, 246）
- しかし、types/index.ts の ProjectAssignment インターフェースに `workload` が定義されていない

**推奨事項**: 型定義を更新

```typescript
export interface ProjectAssignment {
  id: string;
  projectId: string;
  employeeId: string;
  project?: Project;
  employee?: Employee;
  startDate: string;
  endDate?: string;
  role?: string;
  workload?: number;  // 追加が必要
  remark?: string;
  createdAt: string;
  updatedAt: string;
}
```

### Issue #1: 型定義の不一致

**重要度**: 高
**状態**: 未対応

**説明**:
ProjectAssignment インターフェースに `workload` プロパティが定義されていないが、実装では使用されている。

**影響**:
- TypeScriptのコンパイル時に警告が出る可能性
- 型安全性が損なわれる

**修正方法**:
`frontend/src/types/index.ts` の ProjectAssignment インターフェースに以下を追加：

```typescript
workload?: number;
```

**該当ファイル**:
- `frontend/src/types/index.ts` (line 426-439)
- `frontend/src/pages/ProjectDetail.tsx` (line 53, 245-246)
